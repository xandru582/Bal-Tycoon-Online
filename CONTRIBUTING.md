# Contributing to Bal Tycoon Online

Thanks for wanting to contribute! This project is fully open and community-driven.

## Ways to contribute

- **Bug reports** — open an issue with steps to reproduce
- **Feature ideas** — open an issue before building, so we can align
- **Pull requests** — bug fixes, features, docs, tests, balance changes
- **Translations** — help localize the game to your language
- **Game design** — balance feedback, economy suggestions, new event ideas

## Development setup

See the [README](README.md#-local-development) for local setup instructions.

## Pull request process

1. Fork → branch → commit → PR
2. Branch naming: `feat/`, `fix/`, `docs/`, `chore/`
3. Keep PRs focused — one thing per PR
4. Describe what changed and why in the PR body
5. If your PR touches game balance, include before/after screenshots or numbers

## Commit style

We use [Conventional Commits](https://www.conventionalcommits.org/):

```
feat: add new stock ticker NovaBiotech
fix: credits showing as NaN after session reload
docs: update Docker setup instructions
chore: upgrade Vite to 7.1
```

## Reporting bugs

Please include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Browser / OS / Node version if relevant

## Questions?

Open an issue — there are no stupid questions.
